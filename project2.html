LWC Account
Manager Application

A modern Lightning Web Components–based Salesforce application designed to manage Accounts and Contacts with full CRUD capabilities and seamless REST API integration with external systems.

Project Overview

The LWC Account Manager is a custom Salesforce solution built to simplify Account and Contact management using Lightning Web Components. The application provides an intuitive UI for CRUD operations while securely integrating with external systems using REST APIs.

Business Problem

Users relied on standard Salesforce pages that lacked flexibility and a unified experience for managing Accounts and Contacts. Additionally, there was no structured integration with external systems.

Solution Delivered

  • • Built reusable Lightning Web Components
  • • Implemented CRUD for Accounts & Contacts
  • • Integrated external systems via REST API
  • • Ensured secure, Apex-driven backend logic
